# 🔍 SHBHHBSH - سیستم جامع تخصصی جستجو و شناسایی و کشف و تشخیص واقعی دستگاه های استخراج رمزارز

## Comprehensive Professional Cryptocurrency Mining Device Detection System

### 🌟 ویژگی‌های کلیدی

- **تشخیص چندوجهی**: اسکن صوتی، RF، حرارتی، برق و شبکه
- **پایش لحظه‌ای**: نظارت مداوم با نمایش داده‌های زنده
- **نقشه‌برداری جغرافیایی**: نقشه‌های تعاملی با ردیابی موقعیت دستگاه‌ها
- **تحلیل پیشرفته**: تشخیص امضا با هوش مصنوعی و ارزیابی ریسک
- **رابط کاربری حرفه‌ای**: طراحی مدرن و کاربردی برای استفاده تجاری
- **گزارش‌دهی جامع**: گزارش‌های HTML با تجزیه و تحلیل
- **ذخیره‌سازی پایگاه داده**: پایگاه داده SQLite با تحلیل تاریخی
- **سیستم هشدار**: اعلان‌های لحظه‌ای

### 🚀 شروع سریع

1. **نصب وابستگی‌ها**:
   ```bash
   # ویندوز
   install_SHBHHBSH.bat
   
   # لینوکس/مک
   chmod +x install_SHBHHBSH.sh
   ./install_SHBHHBSH.sh
   ```

2. **اجرای سیستم**:
   ```bash
   # نسخه خط فرمان
   python SHBHHBSH_MAIN.py
   
   # نسخه گرافیکی
   python SHBHHBSH_GUI.py
   
   # راه‌انداز کامل
   python SHBHHBSH_LAUNCHER.py
   ```

3. **اجرای سریع**:
   ```bash
   # ویندوز
   RUN_SHBHHBSH.bat
   
   # لینوکس/مک
   chmod +x RUN_SHBHHBSH.sh
   ./RUN_SHBHHBSH.sh
   ```

### 📋 نمونه‌های استفاده

#### اسکن پایه
```python
from SHBHHBSH_MAIN import MiningDeviceDetector

detector = MiningDeviceDetector()
result = detector.comprehensive_scan((35.6892, 51.3890))
print(f"احتمال ماینر: {result['overall_mining_probability']:.2f}")
```

#### اسکن سفارشی
```python
# اسکن صوتی فقط
result = detector.acoustic_scan(duration=30, location=(35.6892, 51.3890))

# اسکن شبکه
result = detector.network_scan("192.168.1.0/24", "comprehensive")
```

#### تولید گزارش
```python
filename = detector.generate_report()
print(f"گزارش ذخیره شد: {filename}")
```
